# NDVI-GO

## Getting Started

Upload this [file](main.ino) to your arduino board, and open Serial monitor to see the result. <br/>


**Wirring connection** 
<br/>


* IR led  ->  D2 (_digital pin 2_)
* RED led  ->  D3 (_digital pin 3_)
* LDR  ->  A0 (_analog pin A0_)
